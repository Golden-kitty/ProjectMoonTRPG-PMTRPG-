所有**失能角色**在战斗结束后都需要根据场上情况进行死亡检定和额外重创效应的抽取。

这里给出对应情况的参考结果表，**TG**可根据实际情况进行调整。

**战后情况表**


|  |  |  |  |  |
|----|----|----|----|----|
| **战后场上** | **死亡检定** | **额外重创效应** | **恢复情况** | **其他损失** |
| 存在友方 | 1 | 0 | 恢复一半生命值 | 失去携带消耗品 |
| 存在非敌方 | 1 | 1 | 应用一个恢复骰 | 失去携带消耗品、随身钱财 |
| 只存在敌方 | 2 | 1 | 恢复一点生命值 | 失去所有随身物品 |

#### **死亡检定**

无需任何修正值，投掷一枚百面骰，如果结果大于或等于角色等级，角色即可免于死亡。
