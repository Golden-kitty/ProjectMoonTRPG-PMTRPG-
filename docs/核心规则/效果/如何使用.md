一个典型的效果包括三个参数：强度，风险，类别。

 


|          |          |          |          |
|----------|----------|----------|----------|
| ** **    | **强效** | **激进** | **战斗** |
| **效果** | ↑        | ↑        | ↑        |
| ** **    | 强度     | 风险     | 类别     |


 

例如，一个中效稳健生命效果的强度（LERP参数）为4，2，D8，0；而稳健风险对应的D6骰面则是正面1-4+负面5-6。使用时根据骰值查找效果，将LERP参数填入描述即可。
